# 🏷️ 二手闲置交换圈（Swap Circle）

> 基于微信群的校园闲置物品交换小程序，让同学之间以物换物变得简单有趣。

---

## 📋 项目概览

| 项目 | 说明 |
|------|------|
| 产品名称 | 二手闲置交换圈（Swap Circle） |
| 产品形态 | 微信小程序 |
| 目标用户 | 大学生、高中生、校园社群成员 |
| 核心价值 | 基于微信关系链的信任背书，以物换物 / 低价出让 |
| 技术栈 | Taro + Vue3（小程序端）+ 云开发 / 轻量后端 |
| 商业模式 | 免费工具 + 本地生活服务广告 |

## 📌 核心功能

- 📸 拍照发布闲置物品
- 🔄 以物换物 / 低价出让双模式
- ❤️ 点「想要」+ 发起交换提议
- 📋 心愿清单 —— 发布想要的物品，自动匹配
- 👥 基于微信群的交换圈管理
- 🔔 交易进度通知
