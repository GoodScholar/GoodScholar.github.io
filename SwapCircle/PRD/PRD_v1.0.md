# 二手闲置交换圈（Swap Circle）产品需求文档 v1.0

> 文档版本：v1.0 | 创建日期：2026-03-24 | 状态：草稿

---

## 1. 产品概述

### 1.1 产品定位

基于微信生态的**圈子化闲置物品交换**小程序。以**学生群体**为核心切入点，同时天然兼容一切有「熟人圈内闲置流转」需求的社群场景。

> 💡 核心理念：**任何有微信群的地方，就能建一个交换圈。**

### 1.2 目标用户

#### 🎯 核心用户（首要优先级）

| 用户群体 | 典型场景 | 核心诉求 |
|----------|---------|----------|
| 大学生 | 同学间交换教材、考研资料、数码配件、生活用品 | 省钱实惠、校内当面交换方便 |
| 毕业季学生 | 毕业离校前大量闲置物品需要快速处理 | 快速出手、不浪费 |
| 社团/班级 | 社团内部交换活动物资、专业工具 | 圈子信任、精准匹配 |
| 考研/考公学生 | 备考资料、辅导书籍的流转 | 低价获取、资料齐全 |

#### 🔄 可扩展用户（产品成熟后逐步覆盖）

| 用户群体 | 典型场景 | 核心诉求 | 扩展时机 |
|----------|---------|----------|----------|
| 小区/社区居民 | 邻里间交换闲置家电、家具、生活用品 | 就近交换、搬家前快速出清 | v2.0 |
| 宝妈/家长群体 | 儿童衣物、玩具、绘本快速流转（孩子长得快，用品淘汰快） | 省钱环保、同龄孩子精准匹配 | v2.0 |
| 兴趣爱好社群 | 手办圈、读书会、骑行群等同好之间交换/转让装备 | 圈内信任、物品懂行 | v2.0 |
| 职场同事圈 | 公司内部闲置数码、办公用品、书籍交换 | 方便快捷、午休当面交换 | v2.5 |
| 培训/考证群体 | CPA/法考/教资等备考资料在考友间流转 | 低成本获取、资料时效性强 | v2.0 |
| 租房群/合租圈 | 搬家时家电家具低价转给同小区新租户 | 就近处理、避免搬运 | v2.5 |

> 💡 **为什么这些群体都适合？** 因为它们有一个共同特征：**已有微信群 + 成员之间有信任基础 + 闲置物品流转需求频繁**。Swap Circle 的「交换圈」模型天然适配所有这类场景。

### 1.3 核心痛点

#### 通用痛点（所有目标群体共有）

1. **闲鱼不适合熟人小圈子** —— 陌生人交易信任成本高，同城交易还要跑很远
2. **微信群直接发闲置** —— 消息秒沉，翻聊天记录找物品太麻烦，体验极差
3. **以物换物没有好工具** —— 现有平台都以「卖」为主，不支持「换」

#### 学生专属痛点

4. **学生预算有限** —— 换比买更实惠，但缺乏交换平台
5. **毕业季物品集中处理** —— 教材、生活用品大量闲置，需要高效的校内流转渠道
6. **教材资料时效性强** —— 上学期的书下学期就有人需要，天然的循环场景

#### 可扩展群体痛点

7. **宝妈群**：孩子长得快，衣物玩具汰换频繁，全买新的太浪费
8. **兴趣社群**：小众物品（手办、黑胶、装备）在大平台难找到同好买家
9. **租房群**：搬家时大件家具不好处理，扔了可惜运走太贵

### 1.4 产品差异化

- ✅ 基于微信关系链的**圈子信任背书**，熟人之间零风险
- ✅ 支持**以物换物**和**低价出让**双模式，灵活适配各场景
- ✅ **心愿清单**功能 —— 发布想要的物品，系统自动匹配
- ✅ 以班级/社团/小区/兴趣群为单位创建交换圈，**精准触达**
- ✅ **毕业季专区** —— 学生场景特色功能，核心增长引擎
- ✅ **一个产品，多场景适配** —— 学生用完毕业后，进入职场/社区/家长群继续用

## 2. 功能模块

### 2.1 交换圈管理

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 创建交换圈 | 输入圈子名称、描述、封面，生成专属小程序码 | P0 |
| 加入交换圈 | 扫码 / 微信群分享链接加入 | P0 |
| 圈子管理 | 圈主可管理成员、审核加入请求、设置公告 | P1 |
| 多圈子支持 | 用户可同时加入多个交换圈（如公司圈 + 小区圈） | P1 |

### 2.2 物品发布

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 拍照发布 | 最多9张图片 + 物品描述 + 分类标签 | P0 |
| 交换模式选择 | 以物换物 / 低价出让 / 免费赠送 | P0 |
| 物品分类 | 教材资料、数码电子、书籍文具、生活用品、服饰箱包、美妆护肤、运动户外、票券卡券、其他 | P0 |
| 物品状态 | 全新 / 几乎全新 / 轻微使用痕迹 / 明显使用痕迹 | P1 |
| 期望交换物 | 可填写希望用什么来换（如"想换一本设计类书籍"） | P1 |

### 2.3 浏览与互动

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 物品流瀑布浏览 | 圈子内闲置物品瀑布流展示 | P0 |
| 点「想要」 | 对心仪物品标记想要 | P0 |
| 发起交换提议 | 选择自己的一件物品，向对方发起换物请求 | P0 |
| 评论 / 私聊 | 就物品细节进行沟通 | P1 |
| 搜索 / 筛选 | 按分类、关键词筛选物品 | P1 |

### 2.4 心愿清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 发布心愿 | "我想要 XXX" —— 告诉圈友你在找什么 | P1 |
| 智能匹配 | 有新物品发布时自动比对心愿清单并提醒 | P2 |
| 心愿墙 | 展示圈内所有心愿，促进物品流通 | P2 |

### 2.5 交易流程

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 交换协商 | 双方确认交换物品和方式 | P0 |
| 交换确认 | 双方确认交换完成 | P0 |
| 交换记录 | 历史交换记录查看 | P1 |
| 评价体系 | 交换完成后互评 | P2 |

### 2.6 个人中心

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 我的发布 | 管理已发布的物品 | P0 |
| 我的交换 | 进行中 / 已完成的交换 | P0 |
| 我的心愿 | 管理心愿清单 | P1 |
| 我的交换圈 | 管理加入的圈子 | P0 |
| 交换数据 | 累计交换次数、环保贡献值 | P2 |

## 7. 用户画像与典型场景

### 画像 A：大二学生小林

- **背景**：大二计算机专业，预算有限
- **场景**：这学期不再需要高数课本，但下学期要学数据结构，想用高数书换一本二手数据结构教材
- **行为路径**：打开小程序 → 发布高数课本（模式：以物换物）→ 填写期望交换物 → 等待同学提议 → 约在图书馆门口交换

### 画像 B：大四毕业生小王

- **背景**：大四即将毕业，宿舍清理出一堆闲置
- **场景**：台灯、键盘、考研真题、衣架等一大堆东西，不想扔但也带不走，想快速转给学弟学妹
- **行为路径**：一键发布多件物品 → 标记「毕业清仓」→ 分享到班级群/院系群 → 学弟学妹们来挑选 → 约楼下当面交还

### 画像 C：社团负责人小张

- **背景**：摄影社社长，社团有一批活动物资需要流转
- **场景**：上届社团买的三脚架、反光板等器材需要传给新成员，但微信群里发了容易被刷走
- **行为路径**：创建「摄影社交换圈」→ 发布器材清单 → 社员加入圈子浏览 → 有需要的社员认领

## 8. 核心业务流程

### 8.1 以物换物流程

```
发布者拍照上传物品
  → 选择「以物换物」模式
  → 填写期望交换物描述
  → 物品上架到交换圈
       ↓
浏览者看到感兴趣的物品
  → 点击「想要」或「发起交换」
  → 选择自己的一件物品作为交换提议
  → 附上留言发送提议
       ↓
发布者收到交换提议通知
  → 查看对方提供的物品
  → 接受 / 拒绝 / 协商
       ↓
双方达成一致
  → 约定交换时间和地点（如食堂/图书馆门口）
  → 当面交换
  → 双方确认交换完成
  → 互相评价（可选）
```

### 8.2 低价出让流程

```
发布者拍照上传物品
  → 选择「低价出让」模式
  → 设置出让价格
  → 物品上架到交换圈
       ↓
浏览者看到感兴趣的物品
  → 点击「想要」联系发布者
  → 双方沟通确认
       ↓
约定交换方式
  → 当面交付 + 微信转账
  → 双方确认完成
```

## 9. 毕业季专区（特色功能）

> 🎓 毕业季（每年5-7月）是校园闲置物品流转的**超级爆发期**，这一场景是 Swap Circle 的核心增长引擎。

### 功能设计

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 毕业清仓模式 | 一键将多件物品标记为「毕业清仓」，批量发布 | P1 |
| 毕业季专区入口 | 首页显眼位置展示毕业季物品合集 | P1 |
| 打包出让 | 支持将多件物品组合为一个「大礼包」出让 | P2 |
| 宿舍清空倒计时 | 显示离校倒计时，营造紧迫感促进交换 | P2 |
| 毕业季排行榜 | 展示「最佳回收人」「环保达人」等趣味榜单 | P2 |

### 运营策略

- 毕业季前1个月启动预热，在各院系群推广
- 设置「毕业季交换达人」勋章，激励毕业生参与
- 新生入学季联动：提前收集新生需求，匹配毕业生物品

## 11. 冷启动方案

### 第一阶段：种子校园（1-2 所高校）

| 策略 | 具体做法 |
|------|---------|
| 校园大使 | 每校招募 3-5 名活跃同学作为圈主，负责创建首批交换圈 |
| 种子内容 | 校园大使先发布 20-30 件物品，让圈子「看起来有东西」|
| 群裂变 | 从班级群/社团群开始推广，「邀请 3 位同学加入解锁心愿功能」|
| 毕业季切入 | 首选 5-7 月毕业季启动，天然的物品流转高峰 |

### 第二阶段：校内扩张

| 策略 | 具体做法 |
|------|---------|
| 院系渗透 | 每个院系建立独立交换圈，院系群推广 |
| 社团合作 | 与学生会、社联合作，成为社团物资交接的官方工具 |
| 线下活动 | 组织「校园交换日」线下活动，配合小程序使用 |

### 第三阶段：多校扩展

| 策略 | 具体做法 |
|------|---------|
| 城市复制 | 先打透一个城市的高校，再跨城市复制 |
| 校园大使体系 | 建立标准化校园大使培训和激励机制 |
| 跨校交换 | 同城高校之间开放跨校交换圈 |

## 12. 风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|---------|
| 违禁物品发布 | 法律风险、平台下架 | 敏感词过滤 + 图片审核（微信内容安全 API）+ 用户举报 |
| 交换纠纷 | 用户流失、口碑受损 | 交换前双方确认机制 + 圈主调解 + 举报处理流程 |
| 虚假物品信息 | 信任破坏 | 实物图片要求 + 校园实名认证 + 不良记录标记 |
| 用户增长瓶颈 | 数据不活跃 | 毕业季运营 + 开学季联动 + 校园大使持续拉新 |
| 假期活跃度下降 | 寒暑假数据断崖 | 假期推出「开学准备清单」提前预约交换 |
| 竞品模仿 | 市场份额被抢 | 深耕「以物换物」差异化 + 校园社群运营壁垒 |

## 13. 核心数据指标

### 北极星指标

> **周活跃交换完成数** —— 衡量产品核心价值是否被持续使用

### 关键指标体系

| 类别 | 指标 | 说明 |
|------|------|------|
| **增长** | 新增用户数 | 日/周/月新注册用户 |
| | 交换圈数量 | 活跃交换圈总数 |
| | 邀请转化率 | 分享链接的打开→注册转化率 |
| **活跃** | DAU / WAU | 日活跃/周活跃用户数 |
| | 发布物品数 | 日均新发布物品数量 |
| | 浏览深度 | 用户平均浏览物品数 |
| **交易** | 交换提议数 | 日均发起交换提议数 |
| | 交换完成率 | 提议→完成交换的转化率 |

| **留存** | 次日留存率 | 注册次日回访比例 |
| | 7日留存率 | 注册7日后回访比起 |
| | 复购率 | 完成一次交换后再次交换的比例 |

## 14. 设计原则

1. **极简优先** —— 学生没耐心，3步内完成发布，1秒内看到结果
2. **信任可见** —— 交换对象的圈子身份、历史评价一目了然
3. **有趣激励** —— 环保贡献值、交换达人勋章让用户有成就感
4. **校园感** —— 设计风格年轻活泼，贴合学生审美，避免商务感
