# 技术方案 —— 二手闲置交换圈（Swap Circle）

> 更新日期：2026-03-24

---

## 1. 技术选型

| 层级 | 技术方案 | 说明 |
|------|---------|------|
| 前端框架 | Taro 3.x + Vue 3 | 跨端小程序开发，兼容微信/支付宝 |
| UI 组件库 | NutUI（京东风格）/ TDesign | 微信小程序适配良好 |
| 状态管理 | Pinia | 轻量高效的状态管理 |
| 后端服务 | 微信云开发 / Node.js + Express | MVP 阶段推荐云开发降低成本 |
| 数据库 | 云开发数据库（MongoDB） | 灵活的文档型数据库 |
| 存储服务 | 云开发存储 | 物品图片存储 |
| 消息推送 | 微信订阅消息 | 交换提醒、心愿匹配通知 |

---

## 2. 核心数据模型

### 2.1 交换圈（Circle）

```javascript
{
  _id: String,           // 圈子 ID
  name: String,          // 圈子名称
  description: String,   // 圈子描述
  coverUrl: String,      // 封面图片
  ownerId: String,       // 圈主 openId
  memberCount: Number,   // 成员数
  memberIds: [String],   // 成员 openId 列表
  qrCodeUrl: String,     // 小程序码
  createdAt: Date,
  updatedAt: Date
}
```

### 2.2 物品（Item）

```javascript
{
  _id: String,           // 物品 ID
  circleId: String,      // 所属圈子
  userId: String,        // 发布者 openId
  title: String,         // 物品标题
  description: String,   // 物品描述
  images: [String],      // 图片 URL 列表（最多9张）
  category: String,      // 分类
  condition: String,     // 成色（全新/几乎全新/轻微使用/明显使用）
  mode: String,          // 交换模式（swap/sell/free）
  price: Number,         // 出让价格（sell 模式）
  wantDescription: String, // 期望交换物描述
  status: String,        // 状态（available/reserved/traded/removed）
  wantCount: Number,     // 想要数
  createdAt: Date,
  updatedAt: Date
}
```

### 2.3 交换提议（SwapProposal）

```javascript
{
  _id: String,
  itemId: String,        // 想要的物品
  proposerId: String,    // 提议人 openId
  offerItemId: String,   // 用来交换的物品（swap 模式）
  offerPrice: Number,    // 出价（sell 模式）
  message: String,       // 留言
  status: String,        // 状态（pending/accepted/rejected/completed）
  createdAt: Date
}
```

### 2.4 心愿（Wish）

```javascript
{
  _id: String,
  circleId: String,
  userId: String,
  description: String,   // 想要什么
  category: String,
  status: String,        // active / fulfilled
  createdAt: Date
}
```

## 4. 关键技术点

### 4.1 图片处理

- 前端压缩：上传前使用 Canvas 压缩至 500KB 以内
- CDN 加速：云存储自带 CDN，配合图片裁剪参数按需加载

### 4.2 内容安全

- 调用微信内容安全接口，审核图片和文字
- 敏感词过滤 + 人工举报机制

### 4.3 消息通知

- 利用微信订阅消息推送关键节点通知
- 交换提议通知、心愿匹配通知、交换完成通知
