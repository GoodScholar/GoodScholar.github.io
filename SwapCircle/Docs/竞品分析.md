## 2. 竞品优劣势分析

### 2.1 闲鱼

- **优势**：用户量大、品类全、支付宝信用体系
- **劣势**：对学生不友好（陌生人交易多、同校交易还要走快递）、不支持以物换物

### 2.2 校园跳蚤市场小程序

- **优势**：针对单一校园、学生群体
- **劣势**：功能单一（只有买卖）、通常只覆盖单校、缺乏圈子和交换概念

### 2.3 微信/QQ 群直发

- **优势**：零门槛、同学之间信任度高
- **劣势**：信息刷屏秒沉、无法检索、无交易管理、毕业季群里全是转让帖体验极差

## 3. 市场机会

1. **校园二手市场刚需** —— 全国3000+高校、4000万+在校大学生，教材和生活用品流转需求巨大
2. **以物换物赛道空白** —— 学生预算有限，「换」比「买」更有吸引力，但没有好工具
3. **毕业季爆发窗口** —— 每年6-7月毕业季，大量物品集中处理，是天然的增长爆点
4. **微信生态红利** —— 班级群/社团群 + 小程序 = 天然裂变渠道
5. **Z世代环保认同** —— 学生群体对循环利用、可持续消费理念接受度高

---

## 4. 差异化定位

```
闲鱼 = 二手交易的「淘宝」（广而全，陌生人）
Swap Circle = 校园二手交换的「班级群增强版」（小而信，同学间）
```

> **核心差异**：不做全量平台，做校园圈子工具。以班级/社团/宿舍楼为单位，同学之间面对面交换，零信任成本。
