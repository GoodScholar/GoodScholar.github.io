## 📦 核心组件总览

React Native 不使用 HTML 标签，而是提供了一套**原生映射组件**：

| RN 组件 | 对应 Web | 对应 Android | 对应 iOS | 说明 |
|---------|----------|-------------|---------|------|
| `View` | `<div>` | `android.view` | `UIView` | 基础容器 |
| `Text` | `<span>` | `TextView` | `UILabel` | 文本显示 |
| `Image` | `<img>` | `ImageView` | `UIImageView` | 图片 |
| `TextInput` | `<input>` | `EditText` | `UITextField` | 输入框 |
| `ScrollView` | `overflow: scroll` | `ScrollView` | `UIScrollView` | 滚动容器 |
| `Pressable` | `<button>` | — | — | 可按压容器 |
| `FlatList` | — | `RecyclerView` | `UITableView` | 高性能列表 |
| `Switch` | `<input type="checkbox">` | `Switch` | `UISwitch` | 开关 |
| `ActivityIndicator` | — | `ProgressBar` | `UIActivityIndicatorView` | 加载器 |

## 🧱 1. View — 万物皆 View

`View` 是最基础的容器组件，相当于 Web 中的 `<div>`：

```tsx
import { View, StyleSheet } from 'react-native';

// 基础容器
<View style={styles.container}>
  <View style={styles.box} />
</View>

// 嵌套布局
<View style={styles.row}>
  <View style={styles.cell} />
  <View style={styles.cell} />
  <View style={styles.cell} />
</View>

const styles = StyleSheet.create({
  container: {
    flex: 1,
    padding: 16,
  },
  row: {
    flexDirection: 'row',
    gap: 8,
  },
  box: {
    width: 100,
    height: 100,
    backgroundColor: '#6366F1',
    borderRadius: 12,
  },
  cell: {
    flex: 1,
    height: 80,
    backgroundColor: '#E0E7FF',
    borderRadius: 8,
  },
});
```

## 📝 2. Text — 文本必须被包裹

在 RN 中，**所有文字必须放在 `<Text>` 组件内**，否则会报错：

```tsx
import { Text, StyleSheet } from 'react-native';

// ❌ 错误：裸文字
<View>Hello</View>

// ✅ 正确
<View>
  <Text>Hello</Text>
</View>

// 文本嵌套（继承父级样式）
<Text style={styles.title}>
  标题文字
  <Text style={styles.highlight}> 高亮部分</Text>
</Text>

// 文本截断
<Text numberOfLines={2} ellipsizeMode="tail">
  这是一段很长很长的文本，超过两行会自动截断并显示省略号...
</Text>

const styles = StyleSheet.create({
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    color: '#1a1a1a',
  },
  highlight: {
    color: '#6366F1',
    fontWeight: '800',
  },
});
```

### Text 常用属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `numberOfLines` | `number` | 最大行数，超出截断 |
| `ellipsizeMode` | `head/middle/tail/clip` | 截断方式 |
| `selectable` | `boolean` | 是否可选中复制 |
| `onPress` | `function` | 点击事件 |
| `adjustsFontSizeToFit` | `boolean` | 自动缩小字号以适应容器（iOS） |

## 🖼 3. Image — 图片加载

```tsx
import { Image, StyleSheet } from 'react-native';

// 网络图片（必须指定宽高）
<Image
  source={{ uri: 'https://example.com/photo.jpg' }}
  style={styles.networkImage}
/>

// 本地图片（自动获取尺寸）
<Image
  source={require('../assets/logo.png')}
  style={styles.localImage}
/>

// 圆形头像
<Image
  source={{ uri: avatarUrl }}
  style={styles.avatar}
/>

// 带占位和加载失败处理
<Image
  source={{ uri: imageUrl }}
  style={styles.photo}
  defaultSource={require('../assets/placeholder.png')}
  onError={() => console.log('图片加载失败')}
  resizeMode="cover"  // cover / contain / stretch / center
/>

const styles = StyleSheet.create({
  networkImage: {
    width: 200,
    height: 200,
    borderRadius: 8,
  },
  localImage: {
    width: 100,
    height: 100,
  },
  avatar: {
    width: 60,
    height: 60,
    borderRadius: 30,  // 宽高一半 = 圆形
  },
  photo: {
    width: '100%',
    height: 200,
  },
});
```

> ⚠️ **注意**：网络图片**必须显式设置宽高**，否则不会显示！本地图片可自动获取尺寸。

## ⌨️ 4. TextInput — 输入框

```tsx
import { TextInput, StyleSheet, useState } from 'react-native';

export default function InputDemo() {
  const [text, setText] = useState('');
  const [password, setPassword] = useState('');

  return (
    <View style={styles.form}>
      {/* 普通输入 */}
      <TextInput
        style={styles.input}
        placeholder="请输入用户名"
        placeholderTextColor="#999"
        value={text}
        onChangeText={setText}
        autoCapitalize="none"  // 关闭自动大写
      />

      {/* 密码输入 */}
      <TextInput
        style={styles.input}
        placeholder="请输入密码"
        placeholderTextColor="#999"
        value={password}
        onChangeText={setPassword}
        secureTextEntry  // 密码模式
      />

      {/* 多行文本 */}
      <TextInput
        style={[styles.input, styles.textArea]}
        placeholder="请输入简介..."
        multiline
        numberOfLines={4}
        textAlignVertical="top"  // Android 多行文字靠上
      />

      {/* 数字键盘 */}
      <TextInput
        style={styles.input}
        placeholder="请输入手机号"
        keyboardType="phone-pad"
        maxLength={11}
      />
    </View>
  );
}

const styles = StyleSheet.create({
  form: { padding: 16, gap: 12 },
  input: {
    borderWidth: 1,
    borderColor: '#E5E7EB',
    borderRadius: 8,
    paddingHorizontal: 12,
    paddingVertical: 10,
    fontSize: 16,
    color: '#1a1a1a',
    backgroundColor: '#F9FAFB',
  },
  textArea: {
    minHeight: 100,
  },
});
```

### keyboardType 键盘类型

| 值 | 效果 |
|---|------|
| `default` | 默认键盘 |
| `number-pad` | 纯数字 |
| `decimal-pad` | 数字 + 小数点 |
| `phone-pad` | 电话键盘 |
| `email-address` | 带 @ 的键盘 |
| `url` | 带 .com 的键盘 |

## 📜 5. ScrollView — 滚动容器

RN 中的 View **默认不可滚动**，超出部分会被裁切。需要滚动时使用 ScrollView：

```tsx
import { ScrollView, View, Text, StyleSheet, RefreshControl } from 'react-native';

export default function ScrollDemo() {
  const [refreshing, setRefreshing] = useState(false);

  const onRefresh = useCallback(async () => {
    setRefreshing(true);
    await fetchData();
    setRefreshing(false);
  }, []);

  return (
    <ScrollView
      style={styles.container}
      contentContainerStyle={styles.content}
      showsVerticalScrollIndicator={false}  // 隐藏滚动条
      // 下拉刷新
      refreshControl={
        <RefreshControl refreshing={refreshing} onRefresh={onRefresh} />
      }
    >
      {Array.from({ length: 20 }).map((_, i) => (
        <View key={i} style={styles.card}>
          <Text style={styles.cardText}>卡片 {i + 1}</Text>
        </View>
      ))}
    </ScrollView>
  );
}

// 横向滚动
<ScrollView
  horizontal                          // 横向滚动
  showsHorizontalScrollIndicator={false}
  contentContainerStyle={{ gap: 12 }}
>
  {tags.map(tag => (
    <View key={tag} style={styles.tag}>
      <Text>{tag}</Text>
    </View>
  ))}
</ScrollView>
```

> ⚠️ **注意**：`ScrollView` 会一次性渲染所有子组件，数据量大时优先使用 `FlatList`。

## 👆 6. Pressable — 可按压交互

`Pressable` 是最灵活的触摸组件，可以自定义按下时的样式：

```tsx
import { Pressable, Text, StyleSheet } from 'react-native';

// 基础按钮
<Pressable
  style={({ pressed }) => [
    styles.button,
    pressed && styles.buttonPressed,  // 按下时的样式
  ]}
  onPress={() => console.log('点击了')}
  onLongPress={() => console.log('长按了')}
>
  {({ pressed }) => (
    <Text style={[styles.btnText, pressed && { opacity: 0.8 }]}>
      {pressed ? '按下中...' : '点击我'}
    </Text>
  )}
</Pressable>

const styles = StyleSheet.create({
  button: {
    backgroundColor: '#6366F1',
    paddingHorizontal: 24,
    paddingVertical: 12,
    borderRadius: 8,
    alignItems: 'center',
  },
  buttonPressed: {
    backgroundColor: '#4F46E5',
    transform: [{ scale: 0.98 }],
  },
  btnText: {
    color: '#fff',
    fontSize: 16,
    fontWeight: '600',
  },
});
```

## 🎨 7. 样式系统 — StyleSheet

### 样式定义方式

```tsx
// 方式一：StyleSheet.create（推荐，有性能优化）
const styles = StyleSheet.create({
  container: { flex: 1, padding: 16 },
});

// 方式二：内联对象（适合动态样式）
<View style={{ backgroundColor: isActive ? '#6366F1' : '#ccc' }} />

// 方式三：样式合并（数组）
<View style={[styles.base, styles.active, { marginTop: 10 }]} />
```

### 常用样式属性速查

```tsx
const styles = StyleSheet.create({
  // 📦 尺寸
  box: {
    width: 100,              // 固定宽
    height: 100,             // 固定高
    minWidth: 50,            // 最小宽
    maxHeight: 200,          // 最大高
    aspectRatio: 16 / 9,     // 宽高比
  },

  // 🎨 颜色与背景
  colored: {
    backgroundColor: '#6366F1',
    opacity: 0.9,
  },

  // 📐 边框
  bordered: {
    borderWidth: 1,
    borderColor: '#E5E7EB',
    borderRadius: 12,
    // 单边圆角
    borderTopLeftRadius: 12,
    borderTopRightRadius: 12,
  },

  // 📏 间距
  spaced: {
    margin: 16,              // 外边距
    marginHorizontal: 16,    // 左右外边距
    marginVertical: 8,       // 上下外边距
    padding: 12,             // 内边距
    paddingHorizontal: 16,
  },

  // 🔤 文字
  text: {
    fontSize: 16,
    fontWeight: 'bold',      // '100'-'900' / 'bold' / 'normal'
    color: '#1a1a1a',
    textAlign: 'center',
    lineHeight: 24,
    letterSpacing: 0.5,
    textDecorationLine: 'underline',
    textTransform: 'uppercase',
  },

  // 🌓 阴影（iOS）
  shadowIOS: {
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.1,
    shadowRadius: 8,
  },

  // 🌓 阴影（Android）
  shadowAndroid: {
    elevation: 4,
  },
});
```

## 📐 8. Flexbox 布局

React Native **默认使用 Flexbox 布局**，且默认 `flexDirection: 'column'`（竖向排列），这和 Web 的默认横向不同！

### 常用布局模式

```tsx
// 纵向居中
<View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
  <Text>居中内容</Text>
</View>

// 横向排列
<View style={{ flexDirection: 'row', gap: 8 }}>
  <View style={{ flex: 1, height: 60, backgroundColor: '#red' }} />
  <View style={{ flex: 2, height: 60, backgroundColor: '#blue' }} />
</View>

// 两端对齐
<View style={{ flexDirection: 'row', justifyContent: 'space-between' }}>
  <Text>左边</Text>
  <Text>右边</Text>
</View>

// 底部固定按钮
<View style={{ flex: 1 }}>
  <ScrollView style={{ flex: 1 }}>
    {/* 页面内容 */}
  </ScrollView>
  <View style={{ padding: 16 }}>
    <Pressable style={styles.fixedBtn}>
      <Text>底部按钮</Text>
    </Pressable>
  </View>
</View>
```

### Flexbox 速查表

| 属性 | 值 | 说明 |
|------|---|------|
| `flexDirection` | `column` / `row` | 主轴方向（默认 column） |
| `justifyContent` | `flex-start/center/flex-end/space-between/space-around/space-evenly` | 主轴对齐 |
| `alignItems` | `flex-start/center/flex-end/stretch/baseline` | 交叉轴对齐 |
| `alignSelf` | 同上 | 单个子元素覆盖 |
| `flex` | `number` | 弹性伸缩比例 |
| `flexWrap` | `wrap/nowrap` | 是否换行 |
| `gap` | `number` | 子元素间距 |
| `position` | `relative/absolute` | 定位方式 |
