## 📊 为什么不直接用 ScrollView？

```
ScrollView: 一次性渲染所有子组件
├── 100 条数据 → 创建 100 个组件 → 占用大量内存
├── 滚动时全部参与布局计算
└── 数据量大时严重卡顿

FlatList: 按需渲染可见区域
├── 100 条数据 → 只渲染屏幕可见的 10+ 条
├── 滚出屏幕的自动回收
└── 性能稳定，万条数据也流畅
```

| 方案 | 适用场景 | 性能 |
|------|---------|------|
| ScrollView | 内容少（< 20 项），不需优化 | ⭐⭐ |
| FlatList | 长列表、动态数据（推荐） | ⭐⭐⭐⭐⭐ |
| SectionList | 带分组头的列表 | ⭐⭐⭐⭐⭐ |
| FlashList | 极致性能（Shopify 出品） | ⭐⭐⭐⭐⭐ |

## 📋 1. FlatList 基础用法

```tsx
import { FlatList, View, Text, StyleSheet } from 'react-native';

interface Item {
  id: string;
  title: string;
  desc: string;
}

const DATA: Item[] = Array.from({ length: 50 }, (_, i) => ({
  id: `${i}`,
  title: `标题 ${i + 1}`,
  desc: `这是第 ${i + 1} 条数据的描述信息`,
}));

export default function ListDemo() {
  return (
    <FlatList
      data={DATA}
      keyExtractor={(item) => item.id}
      renderItem={({ item, index }) => (
        <View style={styles.card}>
          <Text style={styles.title}>{item.title}</Text>
          <Text style={styles.desc}>{item.desc}</Text>
        </View>
      )}
      // 列表分隔线
      ItemSeparatorComponent={() => <View style={styles.separator} />}
      // 列表头部
      ListHeaderComponent={() => (
        <Text style={styles.header}>共 {DATA.length} 条数据</Text>
      )}
      // 列表为空时
      ListEmptyComponent={() => (
        <View style={styles.empty}>
          <Text>暂无数据 📭</Text>
        </View>
      )}
      // 列表尾部
      ListFooterComponent={() => (
        <Text style={styles.footer}>— 到底啦 —</Text>
      )}
    />
  );
}

const styles = StyleSheet.create({
  card: {
    padding: 16,
    backgroundColor: '#fff',
  },
  title: {
    fontSize: 16,
    fontWeight: '600',
    color: '#1a1a1a',
  },
  desc: {
    fontSize: 14,
    color: '#666',
    marginTop: 4,
  },
  separator: {
    height: 1,
    backgroundColor: '#F3F4F6',
  },
  header: {
    padding: 16,
    fontSize: 14,
    color: '#999',
  },
  footer: {
    padding: 24,
    textAlign: 'center',
    color: '#999',
  },
  empty: {
    padding: 60,
    alignItems: 'center',
  },
});
```

## 🔄 2. 下拉刷新 + 上拉加载更多

```tsx
import { FlatList, RefreshControl, ActivityIndicator } from 'react-native';

export default function PaginatedList() {
  const [data, setData] = useState<Item[]>([]);
  const [refreshing, setRefreshing] = useState(false);
  const [loadingMore, setLoadingMore] = useState(false);
  const [hasMore, setHasMore] = useState(true);
  const [page, setPage] = useState(1);

  // 首次加载
  useEffect(() => {
    fetchData(1);
  }, []);

  // 获取数据
  const fetchData = async (pageNum: number) => {
    const res = await api.getList({ page: pageNum, pageSize: 20 });
    if (pageNum === 1) {
      setData(res.list);          // 刷新：替换数据
    } else {
      setData(prev => [...prev, ...res.list]); // 加载更多：追加数据
    }
    setHasMore(res.list.length === 20);
    setPage(pageNum);
  };

  // 下拉刷新
  const onRefresh = useCallback(async () => {
    setRefreshing(true);
    await fetchData(1);
    setRefreshing(false);
  }, []);

  // 上拉加载更多
  const onEndReached = useCallback(async () => {
    if (loadingMore || !hasMore) return;
    setLoadingMore(true);
    await fetchData(page + 1);
    setLoadingMore(false);
  }, [loadingMore, hasMore, page]);

  // 底部加载指示器
  const renderFooter = () => {
    if (loadingMore) {
      return (
        <View style={styles.loadingFooter}>
          <ActivityIndicator size="small" color="#6366F1" />
          <Text style={styles.loadingText}>加载中...</Text>
        </View>
      );
    }
    if (!hasMore && data.length > 0) {
      return <Text style={styles.noMoreText}>— 没有更多了 —</Text>;
    }
    return null;
  };

  return (
    <FlatList
      data={data}
      keyExtractor={(item) => item.id}
      renderItem={({ item }) => <ItemCard item={item} />}
      // 下拉刷新
      refreshControl={
        <RefreshControl
          refreshing={refreshing}
          onRefresh={onRefresh}
          colors={['#6366F1']}        // Android 刷新颜色
          tintColor="#6366F1"          // iOS 刷新颜色
        />
      }
      // 上拉加载更多
      onEndReached={onEndReached}
      onEndReachedThreshold={0.2}  // 距底部 20% 时触发
      ListFooterComponent={renderFooter}
    />
  );
}
```

## 🏷 3. SectionList — 分组列表

适用于通讯录、设置页等分组展示的场景：

```tsx
import { SectionList, View, Text, StyleSheet } from 'react-native';

const SECTIONS = [
  {
    title: '🍎 水果',
    data: ['苹果', '香蕉', '橙子', '草莓'],
  },
  {
    title: '🥬 蔬菜',
    data: ['西兰花', '胡萝卜', '番茄', '黄瓜'],
  },
  {
    title: '🥩 肉类',
    data: ['鸡肉', '牛肉', '猪肉', '鱼肉'],
  },
];

export default function SectionListDemo() {
  return (
    <SectionList
      sections={SECTIONS}
      keyExtractor={(item, index) => item + index}
      // 渲染分组头
      renderSectionHeader={({ section: { title } }) => (
        <View style={styles.sectionHeader}>
          <Text style={styles.sectionTitle}>{title}</Text>
        </View>
      )}
      // 渲染列表项
      renderItem={({ item }) => (
        <View style={styles.item}>
          <Text style={styles.itemText}>{item}</Text>
        </View>
      )}
      // 分组之间的间距
      SectionSeparatorComponent={() => <View style={{ height: 8 }} />}
      // 吸顶效果
      stickySectionHeadersEnabled
    />
  );
}

const styles = StyleSheet.create({
  sectionHeader: {
    backgroundColor: '#F3F4F6',
    paddingHorizontal: 16,
    paddingVertical: 8,
  },
  sectionTitle: {
    fontSize: 16,
    fontWeight: 'bold',
    color: '#374151',
  },
  item: {
    padding: 16,
    backgroundColor: '#fff',
    borderBottomWidth: StyleSheet.hairlineWidth,
    borderBottomColor: '#E5E7EB',
  },
  itemText: {
    fontSize: 16,
    color: '#1a1a1a',
  },
});
```

## ⚡ 4. 性能优化技巧

```tsx
<FlatList
  data={data}
  keyExtractor={(item) => item.id}
  renderItem={renderItem}
  // 🚀 性能优化参数
  initialNumToRender={10}         // 首次渲染数量
  maxToRenderPerBatch={10}        // 每批渲染数量
  windowSize={5}                  // 渲染窗口大小（屏幕倍数）
  removeClippedSubviews={true}    // 裁切不可见视图（Android 效果好）
  getItemLayout={(data, index) => ({  // 已知高度时跳过测量
    length: ITEM_HEIGHT,
    offset: ITEM_HEIGHT * index,
    index,
  })}
/>
```

### 优化 renderItem

```tsx
// ❌ 避免：每次渲染创建新组件
<FlatList
  renderItem={({ item }) => (
    <View>
      <Text>{item.title}</Text>
    </View>
  )}
/>

// ✅ 推荐：使用 React.memo + useCallback
const ItemCard = React.memo(({ item }: { item: Item }) => (
  <View style={styles.card}>
    <Text>{item.title}</Text>
  </View>
));

const renderItem = useCallback(
  ({ item }: { item: Item }) => <ItemCard item={item} />,
  []
);

<FlatList renderItem={renderItem} />
```

### 性能参数速查

| 参数 | 默认值 | 说明 |
|------|-------|------|
| `initialNumToRender` | 10 | 首屏渲染数量 |
| `maxToRenderPerBatch` | 10 | 每次滚动渲染的最大数量 |
| `windowSize` | 21 | 渲染窗口屏幕数（上下各 10 屏） |
| `removeClippedSubviews` | false | 移除屏幕外视图（Android 推荐开） |
| `updateCellsBatchingPeriod` | 50 | 批量更新周期(ms) |

## 🧊 5. 更多列表形态

### 网格布局

```tsx
<FlatList
  data={products}
  numColumns={2}  // 两列网格
  keyExtractor={(item) => item.id}
  columnWrapperStyle={{ gap: 12 }}  // 列间距
  contentContainerStyle={{ padding: 16, gap: 12 }}
  renderItem={({ item }) => (
    <View style={styles.gridItem}>
      <Image source={{ uri: item.image }} style={styles.gridImage} />

      <Text style={styles.price}>¥{item.price}</Text>
    </View>
  )}
/>
```

### 横向列表

```tsx
<FlatList
  data={stories}

  )}

```
