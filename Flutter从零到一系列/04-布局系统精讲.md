# 📐 Flutter 从零到一（四）：布局系统精讲 — 从 CSS 思维到 Flutter 思维

> **系列导读**：认识了 Widget 积木后，本文教你**如何拼装**。
> Flutter 的布局系统看似与 CSS 不同，但核心概念完全相通。

---

## 📊 布局 Widget 速查

| Widget | CSS 等价 | 用途 |
|--------|---------|------|
| `Row` | `display: flex; flex-direction: row` | 水平排列 |
| `Column` | `display: flex; flex-direction: column` | 垂直排列 |
| `Stack` | `position: relative/absolute` | 层叠定位 |
| `Expanded` | `flex: 1` | 占满剩余空间 |
| `Flexible` | `flex: 1` (可缩小) | 弹性空间 |
| `Wrap` | `flex-wrap: wrap` | 自动换行 |
| `SizedBox` | `width` / `height` / `gap` | 固定尺寸/间距 |
| `Padding` | `padding` | 内边距 |
| `Center` | `display: flex; justify-content/align-items: center` | 居中 |
| `Align` | `text-align` / `align-self` | 对齐 |
| `ConstrainedBox` | `max-width` / `min-height` | 约束 |
| `AspectRatio` | `aspect-ratio` | 宽高比 |

## 🔄 1. Row 与 Column — Flex 布局

### Row — 水平排列

```dart
// 基础水平排列
Row(
  children: [
    Icon(Icons.star, color: Colors.amber),
    SizedBox(width: 4),
    Text('4.8'),
    SizedBox(width: 4),
    Text('(128 评价)', style: TextStyle(color: Colors.grey)),
  ],
)

// 主轴对齐（类似 justify-content）
Row(
  mainAxisAlignment: MainAxisAlignment.spaceBetween,  
  // start | center | end | spaceBetween | spaceAround | spaceEvenly
  children: [
    Text('左侧'),
    Text('中间'),
    Text('右侧'),
  ],
)

// 交叉轴对齐（类似 align-items）
Row(
  crossAxisAlignment: CrossAxisAlignment.center,
  // start | center | end | stretch | baseline
  children: [
    Text('小字', style: TextStyle(fontSize: 12)),
    Text('大字', style: TextStyle(fontSize: 32)),
    Text('中字', style: TextStyle(fontSize: 20)),
  ],
)
```

### Column — 垂直排列

```dart
// 常见的卡片内容布局
Column(
  crossAxisAlignment: CrossAxisAlignment.start,  // 左对齐
  children: [
    Text('文章标题', style: TextStyle(fontSize: 20, fontWeight: FontWeight.bold)),
    SizedBox(height: 8),
    Text('2024 年 3 月 17 日', style: TextStyle(color: Colors.grey, fontSize: 12)),
    SizedBox(height: 12),
    Text('这是文章的摘要内容，简要描述文章的核心观点...'),
    SizedBox(height: 16),
    Row(
      children: [
        Icon(Icons.thumb_up, size: 16, color: Colors.grey),
        SizedBox(width: 4),
        Text('128', style: TextStyle(color: Colors.grey)),
        Spacer(),
        TextButton(onPressed: () {}, child: Text('阅读更多')),
      ],
    ),
  ],
)
```

### mainAxisSize — 控制主轴尺寸

```dart
// 默认：占满整个可用空间
Column(
  mainAxisSize: MainAxisSize.max,  // 默认值
  children: [Text('内容')],
)

// 仅包裹内容大小（常用于 Dialog、BottomSheet）
Column(
  mainAxisSize: MainAxisSize.min,  // 只占内容大小
  children: [Text('紧凑内容')],
)
```

## 📏 2. Expanded 与 Flexible — 弹性布局

### Expanded — 占满剩余空间

```dart
// 搜索栏布局：图标 + 弹性输入框 + 按钮
Row(
  children: [
    Icon(Icons.search, color: Colors.grey),
    SizedBox(width: 8),
    Expanded(              // 占满 Row 的剩余空间
      child: TextField(
        decoration: InputDecoration(
          hintText: '搜索...',
          border: InputBorder.none,
        ),
      ),
    ),
    SizedBox(width: 8),
    ElevatedButton(onPressed: () {}, child: Text('搜索')),
  ],
)

// 比例分配（类似 CSS flex: 2 / flex: 1）
Row(
  children: [
    Expanded(
      flex: 2,  // 占 2 份
      child: Container(color: Colors.blue, height: 100),
    ),
    SizedBox(width: 8),
    Expanded(
      flex: 1,  // 占 1 份
      child: Container(color: Colors.red, height: 100),
    ),
  ],
)
```

### fill_width 常见模式

```dart
// 按钮占满宽度
SizedBox(
  width: double.infinity,    // 占满父容器宽度
  child: ElevatedButton(
    onPressed: () {},
    child: Text('全宽按钮'),
  ),
)
```

## 📚 3. Stack 与 Positioned — 层叠布局

```dart
// 基础层叠：图片上方放文字
Stack(
  children: [
    // 底层 — 图片
    ClipRRect(
      borderRadius: BorderRadius.circular(16),
      child: Image.network(
        'https://picsum.photos/400/200',
        width: double.infinity,
        height: 200,
        fit: BoxFit.cover,
      ),
    ),

    // 渐变遮罩
    Positioned.fill(
      child: Container(
        decoration: BoxDecoration(
          borderRadius: BorderRadius.circular(16),
          gradient: LinearGradient(
            begin: Alignment.topCenter,
            end: Alignment.bottomCenter,
            colors: [Colors.transparent, Colors.black87],
          ),
        ),
      ),
    ),

    // 上层 — 文字（左下角定位）
    Positioned(
      bottom: 16,
      left: 16,
      right: 16,
      child: Column(
        crossAxisAlignment: CrossAxisAlignment.start,
        children: [
          Text('风景如画',
            style: TextStyle(color: Colors.white, fontSize: 20, fontWeight: FontWeight.bold)),
          SizedBox(height: 4),
          Text('2024年春天拍摄于杭州西湖',
            style: TextStyle(color: Colors.white70, fontSize: 12)),
        ],
      ),
    ),

    // 右上角 — 收藏按钮
    Positioned(
      top: 12,
      right: 12,
      child: CircleAvatar(
        backgroundColor: Colors.black38,
        child: IconButton(
          icon: Icon(Icons.favorite_border, color: Colors.white),
          onPressed: () {},
        ),
      ),
    ),
  ],
)
```

### 角标 / 徽章


## 🔃 4. Wrap — 自动换行布局

```dart
// 标签云
Wrap(
  spacing: 8,     // 水平间距
  runSpacing: 8,  // 行间距
  children: [
    _buildTag('Flutter'),
    _buildTag('Dart'),
    _buildTag('跨端开发'),
    _buildTag('移动端'),
    _buildTag('状态管理'),
    _buildTag('动画'),
    _buildTag('性能优化'),
    _buildTag('React Native'),
  ],
)

Widget _buildTag(String text) {
  return Chip(
    label: Text(text, style: TextStyle(fontSize: 12)),
    backgroundColor: Colors.indigo.withValues(alpha: 0.1),
    side: BorderSide.none,
    shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(20)),
  );
}
```

## 📱 5. 实战布局案例

### 案例一：用户信息卡片

```dart
class UserCard extends StatelessWidget {
  const UserCard({super.key});

  @override
  Widget build(BuildContext context) {
    return Card(
      margin: const EdgeInsets.all(16),
      shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(16)),
      child: Padding(
        padding: const EdgeInsets.all(16),
        child: Row(
          children: [
            // 头像
            CircleAvatar(
              radius: 30,
              backgroundImage: NetworkImage('https://picsum.photos/100'),
            ),

            const SizedBox(width: 16),

            // 用户信息（Expanded 占满剩余空间）
            Expanded(
              child: Column(
                crossAxisAlignment: CrossAxisAlignment.start,
                children: [
                  Text('张三', style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold)),
                  SizedBox(height: 4),
                  Text('Flutter 高级工程师', style: TextStyle(color: Colors.grey)),
                  SizedBox(height: 8),
                  // 标签行
                  Wrap(
                    spacing: 6,
                    children: [
                      _miniTag('Flutter'),
                      _miniTag('Dart'),
                      _miniTag('iOS'),
                    ],
                  ),
                ],
              ),
            ),

            // 右侧按钮
            OutlinedButton(
              onPressed: () {},
              child: const Text('关注'),
            ),
          ],
        ),
      ),
    );
  }

  Widget _miniTag(String text) {
    return Container(
      padding: EdgeInsets.symmetric(horizontal: 8, vertical: 2),
      decoration: BoxDecoration(
        color: Colors.blue.withValues(alpha: 0.1),
        borderRadius: BorderRadius.circular(4),
      ),
      child: Text(text, style: TextStyle(fontSize: 11, color: Colors.blue)),
    );
  }
}
```

```dart
            _buildSettingItem(Icons.text_fields, '字体大小', trailing: Text('标准', style: TextStyle(color: Colors.grey))),
          ]),

          _buildSection('隐私', [
            _buildSettingItem(Icons.lock, '隐私设置'),
            _buildSettingItem(Icons.security, '账号安全'),
            _buildSettingItem(Icons.notifications, '通知设置'),
          ]),

          _buildSection('关于', [
            _buildSettingItem(Icons.info, '关于我们'),
            _buildSettingItem(Icons.description, '用户协议'),
            _buildSettingItem(Icons.star, '给个好评'),
          ]),

          // 退出按钮
          Padding(
            padding: const EdgeInsets.all(20),
            child: SizedBox(
              width: double.infinity,
              child: OutlinedButton(
                onPressed: () {},
                style: OutlinedButton.styleFrom(foregroundColor: Colors.red),
                child: Text('退出登录'),
              ),
            ),
          ),
        ],
      ),
    );
  }

  Widget _buildSection(String title, List<Widget> children) {
    return Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        Padding(
          padding: const EdgeInsets.fromLTRB(16, 20, 16, 8),
          child: Text(title, style: TextStyle(color: Colors.grey, fontSize: 13)),
        ),
        ...children,
      ],
    );
  }

  Widget _buildSettingItem(IconData icon, String title, {Widget? trailing}) {
    return ListTile(
      leading: Icon(icon),
      title: Text(title),
      trailing: trailing ?? Icon(Icons.chevron_right, color: Colors.grey),
      onTap: () {},
    );
  }
}
```

## 🧩 6. 响应式布局

```dart
class ResponsiveLayout extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return LayoutBuilder(
      builder: (context, constraints) {
        // 根据宽度调整布局
        if (constraints.maxWidth > 900) {
          return _buildWideLayout();     // 桌面端：三栏布局
        } else if (constraints.maxWidth > 600) {
          return _buildMediumLayout();   // 平板：两栏布局
        } else {
          return _buildNarrowLayout();   // 手机：单栏布局
        }
      },
    );
  }

  // 获取屏幕信息
  Widget _buildAdaptive(BuildContext context) {
    final screenWidth = MediaQuery.of(context).size.width;
    final screenHeight = MediaQuery.of(context).size.height;
    final padding = MediaQuery.of(context).padding;  // 安全区域
    final orientation = MediaQuery.of(context).orientation;

    final crossAxisCount = screenWidth > 600 ? 3 : 2;

    return GridView.builder(
      gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
        crossAxisCount: crossAxisCount,  // 根据屏幕宽度调整列数
        crossAxisSpacing: 8,
        mainAxisSpacing: 8,
      ),
      itemBuilder: (_, i) => Card(child: Center(child: Text('Item $i'))),
    );
  }
}
```
