# 💾 Flutter 从零到一（十一）：本地存储与持久化 — 让数据留下来

> **系列导读**：用户的登录状态、设置偏好、收藏数据不能每次打开 App 都消失。
> 本文教你三种本地存储方案，按场景选择最合适的。

---

## 📊 存储方案对比

| 方案 | 适合数据 | 性能 | 复杂度 | 典型场景 |
|------|---------|------|--------|---------|
| SharedPreferences | 简单键值对 | 🟢 快 | ⭐ | Token、设置、开关 |
| Hive CE | 结构化对象 | 🟢 极快 | ⭐⭐ | 用户信息、缓存、收藏 |
| SQLite (sqflite) | 关系型数据 | 🟡 中 | ⭐⭐⭐ | 聊天记录、大量结构化数据 |

## 📦 1. SharedPreferences — 简单键值存储

```yaml
dependencies:
  shared_preferences: ^2.3.0
```

```dart
import 'package:shared_preferences/shared_preferences.dart';

class PrefsService {
  static late SharedPreferences _prefs;

  // 初始化（在 main 中调用一次）
  static Future<void> init() async {
    _prefs = await SharedPreferences.getInstance();
  }

  // Token 管理
  static String? get token => _prefs.getString('auth_token');
  static Future<void> setToken(String token) => _prefs.setString('auth_token', token);
  static Future<void> clearToken() => _prefs.remove('auth_token');
  static bool get isLoggedIn => token != null;

  // 主题设置
  static bool get isDarkMode => _prefs.getBool('dark_mode') ?? false;
  static Future<void> setDarkMode(bool value) => _prefs.setBool('dark_mode', value);

  // 首次启动
  static bool get isFirstLaunch => _prefs.getBool('first_launch') ?? true;
  static Future<void> setFirstLaunchDone() => _prefs.setBool('first_launch', false);

  // 清除所有数据
  static Future<void> clearAll() => _prefs.clear();
}

// main.dart 初始化
void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await PrefsService.init();
  runApp(const ProviderScope(child: MyApp()));
}

// 使用
if (PrefsService.isFirstLaunch) {
  // 显示引导页
  PrefsService.setFirstLaunchDone();
}
```

## 📦 2. Hive CE — 高性能对象存储

```yaml
dependencies:
  hive_ce: ^2.9.0
  hive_ce_flutter: ^2.2.0
dev_dependencies:
  hive_ce_generator: ^1.7.0
  build_runner: ^2.4.13
```

```dart
// 定义数据模型
import 'package:hive_ce/hive.dart';
part 'favorite_item.g.dart';

@HiveType(typeId: 0)
class FavoriteItem extends HiveObject {
  @HiveField(0)
  String id;

  @HiveField(1)
  String title;

  @HiveField(2)
  String imageUrl;

  @HiveField(3)
  DateTime addedAt;

  FavoriteItem({
    required this.id,
    required this.title,
    required this.imageUrl,
    required this.addedAt,
  });
}

// 初始化
Future<void> initHive() async {
  await Hive.initFlutter();
  Hive.registerAdapter(FavoriteItemAdapter());
  await Hive.openBox<FavoriteItem>('favorites');
}

// CRUD 操作
class FavoriteService {
  static Box<FavoriteItem> get _box => Hive.box<FavoriteItem>('favorites');

  // 添加收藏
  static Future<void> add(FavoriteItem item) async {
    await _box.put(item.id, item);
  }

  // 取消收藏
  static Future<void> remove(String id) async {
    await _box.delete(id);
  }

  // 是否已收藏
  static bool isFavorited(String id) => _box.containsKey(id);

  // 获取所有收藏
  static List<FavoriteItem> getAll() => _box.values.toList();

  // 收藏数量
  static int get count => _box.length;
}

// 使用
await FavoriteService.add(FavoriteItem(
  id: 'article-1',
  title: 'Flutter 入门指南',
  imageUrl: 'https://example.com/img.jpg',
  addedAt: DateTime.now(),
));

print(FavoriteService.isFavorited('article-1'));  // true
print(FavoriteService.count);  // 1
```

## 📦 3. SQLite — 关系型数据库

```yaml
dependencies:
  sqflite: ^2.3.0
  path: ^1.9.0
```

```dart
import 'package:sqflite/sqflite.dart';
import 'package:path/path.dart';

class DatabaseHelper {
  static Database? _db;

  static Future<Database> get database async {
    _db ??= await _initDB();
    return _db!;
  }

  static Future<Database> _initDB() async {
    final path = join(await getDatabasesPath(), 'app.db');
    return openDatabase(
      path,
      version: 1,
      onCreate: (db, version) async {
        await db.execute('''
          CREATE TABLE messages (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            sender TEXT NOT NULL,
            content TEXT NOT NULL,
            timestamp INTEGER NOT NULL,
            is_read INTEGER DEFAULT 0
          )
        ''');
      },
    );
  }

  // 插入消息
  static Future<int> insertMessage(Map<String, dynamic> message) async {
    final db = await database;
    return db.insert('messages', message);
  }

  // 查询消息
  static Future<List<Map<String, dynamic>>> getMessages({int limit = 50}) async {
    final db = await database;
    return db.query('messages', orderBy: 'timestamp DESC', limit: limit);
  }

  // 标记已读
  static Future<void> markAsRead(int id) async {
    final db = await database;
    await db.update('messages', {'is_read': 1}, where: 'id = ?', whereArgs: [id]);
  }

  // 删除消息
  static Future<void> deleteMessage(int id) async {
    final db = await database;
    await db.delete('messages', where: 'id = ?', whereArgs: [id]);
  }

  // 未读数量
  static Future<int> getUnreadCount() async {
    final db = await database;
    final result = await db.rawQuery('SELECT COUNT(*) as count FROM messages WHERE is_read = 0');
    return result.first['count'] as int;
  }
}
```

## 🎯 选型决策

```
你需要存什么数据？
│
├── Token / 设置 / 开关（键值对） → SharedPreferences
├── 用户信息 / 缓存 / 收藏列表 → Hive CE
└── 聊天记录 / 需要复杂查询 → SQLite
```
