## 🥚 阶段一：setState — 最简单的状态管理

```dart
class CounterPage extends StatefulWidget {
  const CounterPage({super.key});
  @override
  State<CounterPage> createState() => _CounterPageState();
}

class _CounterPageState extends State<CounterPage> {
  int _count = 0;
  
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(child: Text('$_count', style: TextStyle(fontSize: 48))),
      floatingActionButton: FloatingActionButton(
        onPressed: () => setState(() => _count++),
        child: Icon(Icons.add),
      ),
    );
  }
}
```

### setState 的局限

```
场景：购物车数量需要在多个页面显示

页面 A（商品列表）   页面 B（购物车）   页面 C（底部Tab角标）
    ↓                   ↓                  ↓
  都需要知道购物车数量，但 setState 只能管自己的状态！
```

> 当**多个 Widget 需要共享同一个状态**时，setState 就力不从心了。

## 🐣 阶段二：理解状态共享的本质

### 状态提升（Lifting State Up）

```dart
// 把共享状态提升到公共父 Widget
class ParentWidget extends StatefulWidget {
  @override
  State<ParentWidget> createState() => _ParentWidgetState();
}

class _ParentWidgetState extends State<ParentWidget> {
  int _cartCount = 0;

  void _addToCart() {
    setState(() => _cartCount++);
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        // 子组件通过参数接收状态和回调
        ProductList(onAddToCart: _addToCart),
        CartBadge(count: _cartCount),
      ],
    );
  }
}

// 问题：层级很深时，参数要层层传递（Prop Drilling）
// Parent → Child → GrandChild → GreatGrandChild → 才能用到数据
// 这就是为什么需要更好的方案。
```

## 🦅 阶段三：Riverpod — 生产级状态管理

### 安装

```yaml
dependencies:
  flutter_riverpod: ^3.2.1
  riverpod_annotation: ^4.0.2
dev_dependencies:
  riverpod_generator: ^4.0.3
  build_runner: ^2.4.13
```

### 核心概念三件套

```
1. Provider（数据源）→ 声明"数据从哪来"
2. Ref（连接器）   → 读取和操作数据
3. Consumer（消费）→ Widget 监听数据变化
```

### 第一步：包裹根组件

```dart
void main() {
  runApp(
    const ProviderScope(  // Riverpod 的根
      child: MyApp(),
    ),
  );
}
```

### 第二步：创建 Provider

```dart
// lib/providers/counter_provider.dart
import 'package:riverpod_annotation/riverpod_annotation.dart';
part 'counter_provider.g.dart';

// 简单状态
@riverpod
class Counter extends _$Counter {
  @override
  int build() => 0;  // 初始值

  void increment() => state++;
  void decrement() => state--;
  void reset() => state = 0;
}

// 购物车状态
@riverpod
class Cart extends _$Cart {
  @override
  List<CartItem> build() => [];

  void addItem(CartItem item) {
    state = [...state, item];
  }

  void removeItem(String id) {
    state = state.where((item) => item.id != id).toList();
  }

  // 派生数据
  int get totalCount => state.length;
  double get totalPrice => state.fold(0, (sum, item) => sum + item.price);
}
```

### 第三步：在 UI 中使用

```dart
// 方式一：ConsumerWidget（整个 Widget 监听）
class CounterPage extends ConsumerWidget {
  const CounterPage({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final count = ref.watch(counterProvider);         // 监听状态
    final notifier = ref.read(counterProvider.notifier); // 获取操作器

    return Scaffold(
      body: Center(
        child: Text('$count', style: TextStyle(fontSize: 48)),
      ),
      floatingActionButton: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          FloatingActionButton(
            heroTag: 'add',
            onPressed: notifier.increment,
            child: Icon(Icons.add),
          ),
          SizedBox(height: 8),
          FloatingActionButton(
            heroTag: 'sub',
            onPressed: notifier.decrement,
            child: Icon(Icons.remove),
          ),
        ],
      ),
    );
  }
}

// 方式二：Consumer（局部监听，减少重建）
class ProductPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('商品页'),
        actions: [
          // 只有角标部分监听购物车状态
          Consumer(
            builder: (context, ref, _) {
              final count = ref.watch(cartProvider).length;
              return Badge(
                label: Text('$count'),
                isLabelVisible: count > 0,
                child: IconButton(
                  icon: Icon(Icons.shopping_cart),
                  onPressed: () => context.push('/cart'),
                ),
              );
            },
          ),
        ],
      ),
      body: Text('商品列表'),  // 这部分不会因为购物车变化而重建
    );
  }
}
```

### ref.watch vs ref.read vs ref.listen

```dart
class DemoPage extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    // ✅ ref.watch — 在 build 方法中使用，状态变化自动重建
    final count = ref.watch(counterProvider);

    // ✅ ref.listen — 监听变化执行副作用（弹窗、跳转等）
    ref.listen(counterProvider, (prev, next) {
      if (next == 10) {
        ScaffoldMessenger.of(context).showSnackBar(
          SnackBar(content: Text('恭喜！达到 10 次！')),
        );
      }
    });

    return ElevatedButton(
      onPressed: () {
        // ✅ ref.read — 在回调中使用，只读取一次，不订阅
        ref.read(counterProvider.notifier).increment();
      },
      child: Text('当前: $count'),
    );
  }
}
```

| 方法 | 用在哪里 | 是否订阅 | 用途 |
|------|---------|---------|------|
| `ref.watch` | `build()` 方法中 | ✅ 是 | 读取 + 监听变化 |
| `ref.read` | 回调 / 事件处理中 | ❌ 否 | 只读取一次 |
| `ref.listen` | `build()` 方法中 | ✅ 是 | 监听变化执行副作用 |

### 异步数据

```dart
// 自动管理 loading / data / error 三态
@riverpod
Future<List<Product>> productList(Ref ref) async {
  final dio = ref.watch(dioProvider);
  final response = await dio.get('/api/products');
  return (response.data['list'] as List)
      .map((e) => Product.fromJson(e))
      .toList();
}

// UI 中使用
class ProductListPage extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final productsAsync = ref.watch(productListProvider);

    return productsAsync.when(
      data: (products) => ListView.builder(
        itemCount: products.length,
        itemBuilder: (_, i) => ListTile(title: Text(products[i].name)),
      ),
      loading: () => Center(child: CircularProgressIndicator()),
      error: (err, _) => Center(
        child: Column(
          mainAxisSize: MainAxisSize.min,
          children: [
            Text('加载失败: $err'),
            ElevatedButton(
              onPressed: () => ref.invalidate(productListProvider),
              child: Text('重试'),
            ),
          ],
        ),
      ),
    );
  }
}
```
