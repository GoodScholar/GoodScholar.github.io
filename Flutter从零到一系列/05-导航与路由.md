# 🗺 Flutter 从零到一（五）：导航与路由 — 让页面跳起来

> **系列导读**：一个 App 不可能只有一个页面。本文带你从基础 Navigator 到声明式 GoRouter，
> 掌握页面跳转、参数传递、Tab 导航和抽屉菜单。

## 📊 路由方案对比

| 方案 | 类型 | 适合场景 | 推荐度 |
|------|------|---------|--------|
| `Navigator.push/pop` | 命令式 | 学习理解、简单 App | ⭐⭐ |

## 🚪 1. Navigator 基础（命令式路由）

### 页面跳转与返回

```dart
// 跳转到新页面（push 压入路由栈）
Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => const DetailPage()),
);

// 带参数跳转
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (context) => DetailPage(productId: '123', title: '商品详情'),
  ),
);

// 返回上一页（pop 弹出路由栈）
Navigator.pop(context);

// 带返回值
// 在 DetailPage 中：
Navigator.pop(context, '选择了商品 A');

// 在调用页面接收返回值：
final result = await Navigator.push<String>(
  context,
  MaterialPageRoute(builder: (context) => const SelectPage()),
);
print('返回值: $result');

// 替换当前页面（登录成功后替换登录页）
Navigator.pushReplacement(
  context,
  MaterialPageRoute(builder: (context) => const HomePage()),
);

// 清空路由栈并跳转（退出登录时使用）
Navigator.pushAndRemoveUntil(
  context,
  MaterialPageRoute(builder: (context) => const LoginPage()),
  (route) => false,  // 移除所有旧路由
);
```

### 完整示例：列表 → 详情

```dart
// 列表页
class ProductListPage extends StatelessWidget {
  final List<Map<String, String>> products = [
    {'id': '1', 'name': 'Flutter 实战', 'price': '¥59'},
    {'id': '2', 'name': 'Dart 入门', 'price': '¥39'},
    {'id': '3', 'name': '移动端架构', 'price': '¥79'},
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('商品列表')),
      body: ListView.builder(
        itemCount: products.length,
        itemBuilder: (context, index) {
          final product = products[index];
          return ListTile(
            title: Text(product['name']!),
            subtitle: Text(product['price']!),
            trailing: const Icon(Icons.chevron_right),
            onTap: () async {
              // 跳转到详情页，等待返回值
              final result = await Navigator.push<bool>(
                context,
                MaterialPageRoute(
                  builder: (_) => ProductDetailPage(
                    name: product['name']!,
                    price: product['price']!,
                  ),
                ),
              );
              if (result == true) {
                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(content: Text('已添加 ${product['name']} 到购物车')),
                );
              }
            },
          );
        },
      ),
    );
  }
}

// 详情页
class ProductDetailPage extends StatelessWidget {
  final String name;
  final String price;

  const ProductDetailPage({super.key, required this.name, required this.price});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text(name)),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text(name, style: TextStyle(fontSize: 24, fontWeight: FontWeight.bold)),
            SizedBox(height: 8),
            Text(price, style: TextStyle(fontSize: 20, color: Colors.red)),
            Spacer(),
            SizedBox(
              width: double.infinity,
              child: ElevatedButton(
                onPressed: () => Navigator.pop(context, true),  // 返回 true
                child: Text('加入购物车'),
              ),
            ),
          ],
        ),
      ),
    );
  }
}
```

## 🛣 2. GoRouter（声明式路由 — 生产推荐）

### 安装与配置

```yaml
dependencies:
  go_router: ^14.0.0
```

```dart
// lib/config/router.dart
import 'package:go_router/go_router.dart';

final router = GoRouter(
  initialLocation: '/',
  routes: [
    GoRoute(
      path: '/',
      builder: (context, state) => const HomePage(),
    ),
    GoRoute(
      path: '/product/:id',
      builder: (context, state) {
        final id = state.pathParameters['id']!;
        final name = state.uri.queryParameters['name'] ?? '';
        return ProductDetailPage(productId: id, name: name);
      },
    ),
    GoRoute(
      path: '/settings',
      builder: (context, state) => const SettingsPage(),
    ),
    GoRoute(
      path: '/login',
      builder: (context, state) => const LoginPage(),
    ),
  ],
);

// main.dart 中使用
MaterialApp.router(
  routerConfig: router,
  theme: ThemeData(useMaterial3: true),
);
```

### 导航操作

```dart
// 跳转
context.go('/product/123?name=Flutter实战');    // 替换路由
context.push('/product/123');                   // 压入路由栈

// 返回
context.pop();

// 带路径参数
context.go('/product/${product.id}');

// 带 query 参数
context.go('/search?keyword=flutter&page=1');
```

### ShellRoute — Tab 底部导航

```dart
final router = GoRouter(
  initialLocation: '/home',
  routes: [
    ShellRoute(
      builder: (context, state, child) {
        return MainShell(child: child);  // 共享底部导航
      },
      routes: [
        GoRoute(path: '/home', builder: (_, __) => const HomePage()),
        GoRoute(path: '/discover', builder: (_, __) => const DiscoverPage()),
        GoRoute(path: '/profile', builder: (_, __) => const ProfilePage()),
      ],
    ),
    // 不需要底部导航的页面放在 ShellRoute 外面
    GoRoute(path: '/product/:id', builder: (_, state) => ProductDetailPage(id: state.pathParameters['id']!)),
    GoRoute(path: '/login', builder: (_, __) => const LoginPage()),
  ],
);

// MainShell — 底部导航壳
class MainShell extends StatelessWidget {
  final Widget child;
  const MainShell({super.key, required this.child});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: child,
      bottomNavigationBar: NavigationBar(
        selectedIndex: _getSelectedIndex(context),
        onDestinationSelected: (index) {
          switch (index) {
            case 0: context.go('/home');
            case 1: context.go('/discover');
            case 2: context.go('/profile');
          }
        },
        destinations: const [
          NavigationDestination(icon: Icon(Icons.home), label: '首页'),
          NavigationDestination(icon: Icon(Icons.explore), label: '发现'),
          NavigationDestination(icon: Icon(Icons.person), label: '我的'),
        ],
      ),
    );
  }

  int _getSelectedIndex(BuildContext context) {
    final location = GoRouterState.of(context).matchedLocation;
    if (location.startsWith('/home')) return 0;
    if (location.startsWith('/discover')) return 1;
    if (location.startsWith('/profile')) return 2;
    return 0;
  }
}
```

## 🗂 3. Tab 导航 — TabBar

class CategoryPage extends StatelessWidget {
