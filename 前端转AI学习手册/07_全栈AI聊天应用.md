### SSE 核心代码

```python
from fastapi.responses import StreamingResponse

async def chat_stream(messages: list):
    response = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=messages,
        stream=True,
    )
    for chunk in response:
        content = chunk.choices[0].delta.content
        if content:
            yield f"data: {content}\n\n"

@app.post("/api/chat")
async def chat(request: ChatRequest):
    return StreamingResponse(
        chat_stream(request.messages),
        media_type="text/event-stream",
    )
```

## 🎮 章节小游戏：AI 酒馆 · NPC 对话

构建一个"AI 酒馆"——3 个 NPC 各有人设，玩家通过对话套出秘密情报，集齐 3 段密码通关。

### 游戏设定

```
🍺 欢迎来到「代码酒馆」！

店里有 3 位 NPC：
  1. 🧙 老法师 — 说话文绉绉，喜欢用比喻，知道密码第一部分
  2. 🗡️ 女剑客 — 性格暴躁，夸她的剑法她会透露线索
  3. 🎭 吟游诗人 — 只用押韵方式说话，密码藏在他的诗里

请选择和谁对话（1/2/3），输入 quit 离开酒馆
> 1

🧙 老法师："年轻人，你来这里寻找什么？"
> 我听说你知道一个秘密

🧙 老法师："秘密如同深井中的月影，你需先证明诚意..."
```

### 需要用到的知识点

| 功能 | 知识点 |
|------|--------|
| 3 个 NPC | 3 套 System Prompt |
| 多轮对话 | messages 数组维护 |
| Streaming 回复 | SSE 逐字输出 |
| 前端 Chat UI | 消息气泡 + 切换 NPC |
| 密码验证 | 3 个线索拼出完整密码 |

### 最低要求

- [ ] 3 个 NPC，各有鲜明性格
- [ ] 每个 NPC 持有密码的一部分
- [ ] Streaming 逐字输出
- [ ] 集齐 3 段密码 → 通关
