## 使用方式

1. **按顺序学习**：从 01 开始，每完成一章勾选底部的通关 Checklist
2. **通关往前走**：Checklist 全勾了再进入下一章，不要跳
3. **遇到困难**：先跳过，做项目时再回来补——边做边学
4. **每天投入**：工作日 1.5h，周末 2.5h，共约每周 12-15h
