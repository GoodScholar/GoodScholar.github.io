## 🎮 章节小游戏：AI 模拟面试

构建一个 AI 面试官，模拟真实的 AI 工程师技术面试。面试结束后自动评分+给改进建议。

### 游戏流程

```
🎤 AI 面试官："你好！我是今天的技术面试官。
   我们将进行一场 AI 工程师面试，共 5 道题，涵盖：
   - LLM 基础
   - RAG 系统设计
   - Agent 架构
   - 工程实践
   - 系统设计

   准备好了吗？"
> 准备好了

📋 第 1 题（LLM 基础）：
   "请解释 Temperature 参数的作用。在什么场景下你会把它设为 0？"

> Temperature 控制模型输出的随机性...

📋 第 2 题（RAG）：
   "如果用户问的问题在知识库中找不到相关信息，你会怎么处理？"

> ...

📊 面试评估报告：
╔═══════════════╦═══════╦══════════════════════╗
║ 题目          ║ 得分  ║ 点评                  ║
╠═══════════════╬═══════╬══════════════════════╣
║ LLM 基础      ║ 85/100║ 概念清晰，可补充 top_p ║
║ RAG 设计      ║ 70/100║ 缺少相似度阈值策略     ║
║ Agent 架构    ║ 90/100║ 回答全面              ║
║ 工程实践      ║ 75/100║ 需补充监控和成本控制   ║
║ 系统设计      ║ 80/100║ 整体不错              ║
╠═══════════════╬═══════╬══════════════════════╣
║ 总分          ║ 80/100║ 建议：offer ✅         ║
╚═══════════════╩═══════╩══════════════════════╝
```

### 需要用到的知识点

| 功能 | 知识点 |
|------|--------|
| AI 面试官 | System Prompt 设计 |
| 5 道面试题 | 从题库随机抽取 |
| 多轮追问 | 对话历史管理 |
| 自动评分 | LLM 当评委打分 |
| 评估报告 | 结构化输出 + 表格 |

### 最低要求

- [ ] 题库至少 15 题（每次随机 5 题）
- [ ] AI 面试官能根据回答追问
- [ ] 每题 100 分制自动评分
- [ ] 最终生成评估报告 + 改进建议
