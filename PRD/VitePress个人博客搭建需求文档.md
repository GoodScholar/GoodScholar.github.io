# VitePress 个人博客搭建需求文档

## 背景
将 `/Users/shen/SZG/文章` 目录下现有 70+ 篇技术文章搭建为免费的个人博客网站。

## 技术方案
- **框架**：VitePress（基于 Vue + Vite 的静态站点生成器）
- **托管**：GitHub Pages（完全免费）
- **部署**：GitHub Actions 自动构建部署
- **域名**：`xxx.github.io`（免费）

## 功能需求

- Hero 区域展示博客名称和简介
- Features 展示各文章系列入口
