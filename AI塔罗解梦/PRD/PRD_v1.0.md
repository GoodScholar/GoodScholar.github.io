## 一、产品概述

### 1.1 产品定位

一款基于 AI 大语言模型的**塔罗占卜与梦境解读**微信小程序，为用户提供即时、个性化的心灵探索体验。

**一句话描述**：用 AI 读懂你的牌面和梦境，随时随地获得心灵指引。

### 1.2 产品愿景

成为微信生态中最受欢迎的 AI 心灵探索工具，让每个人都能通过塔罗和解梦获得情绪价值和自我觉察。

### 1.3 为什么是小程序？

| 维度 | 小程序优势 |
|---|---|
| 获客 | 微信生态内裂变传播（群聊、朋友圈），获客成本趋近于零 |
| 转化 | 无需下载，扫码即用，转化率远高于 App Store |
| 支付 | 微信支付零摩擦，付费转化率高 |
| 留存 | **本产品不依赖日留存**，按次付费模型天然适合"用完即走" |

## 二、市场分析

### 2.1 目标市场

- 全球塔罗市场规模：2024 年约 2.3 亿美元，年增长率 8%+
- 国内"泛玄学"小程序用户活跃度高：测测星座、灵机妙算等头部产品月活百万级
- 小红书"塔罗"相关笔记超 **500 万篇**，需求旺盛

### 2.2 竞品分析

| 竞品 | 形态 | 核心功能 | 不足 |
|---|---|---|---|
| 测测星座 | 小程序+App | 星座运势、塔罗测试 | 内容模板化，缺乏个性化深度解读 |
| 灵机妙算 | 小程序+App | 八字、塔罗、风水 | 商业化过重，广告多，体验差 |
| 塔罗牌占卜 | 小程序 | 抽牌+简单解读 | 解读文字粗糙，无 AI 能力 |
| ChatGPT 直接用 | 对话 | 自由对话解读 | 缺乏仪式感和产品化体验 |

### 2.3 我们的差异化

> **"仪式感 × AI 深度 × 社交裂变"三合一**

- **仪式感**：精美的翻牌动画、牌面渲染，而非冷冰冰的文字对话
- **AI 深度**：基于 LLM 的个性化解读，结合用户描述的具体情境，而非千篇一律的模板
- **社交裂变**：每次解读自动生成精美海报，一键分享引发好友好奇

### 4.2 核心功能详细设计

#### 功能一：塔罗占卜（核心付费功能）

**用户流程**：

```
选择问题类型 → 描述困惑 → 选择牌阵 → 洗牌动画 → 逐张翻牌 → AI 生成解读 → 查看完整解读 → 分享海报
```

**牌阵类型**（MVP 阶段）：

| 牌阵 | 牌数 | 适用场景 | 定价 |
|---|---|---|---|
| 单牌速占 | 1 张 | 是/否类简单问题 | 免费（每日1次） |
| 时间之流（三牌阵） | 3 张 | 过去/现在/未来 | ¥3.9 |
| 关系十字 | 5 张 | 双方关系深度分析 | ¥6.9 |
| 凯尔特十字 | 10 张 | 全面深度解读 | ¥9.9 |

**AI 解读要求**：
- 每张牌单独解读（牌面含义 + 位置含义 + 与问题的关联）
- 牌面之间的关联分析
- 综合建议（不少于 300 字）
- 语气要求：温暖、有共情力、具有启发性，**严禁恐吓或绝对化表述**

#### 功能二：AI 解梦（核心付费功能）

**用户流程**：

```
描述梦境（文字/语音） → AI 提取关键象征 → 逐个象征解析 → 整体梦境含义 → 心理学视角 → 分享海报
```

**定价**：¥1.9/次（低价高频策略）

**AI 解读要求**：
- 提取梦境中 3-5 个核心象征物
- 对每个象征物进行荣格心理学视角解读
- 综合分析梦境反映的内在情绪和潜意识
- **严禁做出"预言"式表述**，仅从心理学角度分析

#### 功能三：每日一牌（免费引流功能）

- 每日 0:00 自动更新一张塔罗牌
- 附带简短寄语（50 字以内）
- 所有用户免费查看
- **核心目的**：引导用户每日打开 + 分享裂变

#### 功能四：分享海报（裂变引擎）

每次占卜/解梦完成后自动生成精美海报：
- 包含：牌面图 + 核心解读摘要 + 小程序码
- 风格：暗色系 + 神秘感 + 金色点缀
- 附带引导文案："我刚抽到了【命运之轮】，你也来试试？"
- **海报底部带小程序码**，扫码直接进入

## 五、技术方案

### 5.1 技术栈

| 层级 | 选型 |
|---|---|
| 前端 | 微信小程序原生（WXML + WXSS + JS）或 uni-app |
| 后端 | Node.js / Python（轻量 API 服务） |
| AI 能力 | 大模型 API（DeepSeek / 通义千问 / GPT-4o-mini） |
| 数据库 | MySQL 或 MongoDB（用户记录 + 订单） |
| 图片生成 | Canvas API（海报生成）|
| 部署 | 微信云开发 或 轻量云服务器 |

### 5.2 AI Prompt 工程

**塔罗解读 Prompt 核心框架**：

```
你是一位经验丰富的塔罗解读师，精通韦特塔罗体系。
用户的问题类型：{category}
用户的困惑：{user_question}
抽到的牌阵：{spread_type}
牌面结果：{cards}（含正逆位）

请按以下结构为用户解读：
1. 逐牌解读：每张牌的含义 + 在当前位置的指向
2. 牌面关联：牌与牌之间的能量流动
3. 综合建议：温暖、有启发性的心灵指引

要求：
- 语气温暖有共情力，像一位睿智的朋友
- 严禁恐吓、绝对化表述
- 严禁涉及健康诊断或违法内容
- 解读不少于 300 字
```

### 5.3 成本估算

| 项目 | 成本/次 | 说明 |
|---|---|---|
| AI API 调用（塔罗） | ¥0.02-0.05 | DeepSeek API 价格极低 |
| AI API 调用（解梦） | ¥0.01-0.03 | 输入较短 |
| 云服务器 | ¥50-100/月 | 轻量级即可 |
| 小程序认证 | ¥300/年 | 一次性 |

> **核心结论**：单次占卜 API 成本不到 ¥0.05，定价 ¥3.9-9.9，**毛利率 > 95%**。

### 6.1 收入来源

| 收入类型 | 说明 | 预估占比 |
|---|---|---|
| 付费占卜 | 三牌阵/五牌阵/十牌阵按次收费 | 60% |
| 付费解梦 | 每次 ¥1.9 | 20% |
| 包月/包年卡 | ¥19.9/月，不限次数占卜 | 15% |
| 解锁高级牌阵 | 特殊牌阵（如灵魂使命牌阵）单独售卖 | 5% |

### 6.2 裂变增长策略

```
用户完成占卜 → 分享海报到朋友圈/群聊
                    ↓
          好友扫码进入小程序 → 免费体验单牌占卜
                    ↓
          好奇心驱动 → 付费深度占卜 → 再次分享
                    ↓
                 增长飞轮
```

**具体裂变机制**：
- **首次分享奖励**：分享海报后获得 1 次免费三牌占卜
- **邀请好友奖励**：每成功邀请 1 位好友付费，奖励 1 次免费占卜
- **节日限定活动**：情人节/水逆期间推出限定牌阵，刺激分享

### 6.3 收入预估（保守）

| 阶段 | 日活 | 付费率 | 客单价 | 月收入 |

| 冷启动期（1-2月） | 500 | 8% | ¥5 | ¥6,000 |

- **主色调**：深紫 (#1A0A2E) + 金色 (#D4AF37) + 星空蓝 (#0C1445)

### 7.2 关键页面

1. **首页**：星空背景 + 中央水晶球动画 + 三大功能入口
2. **占卜页**：暗色桌面 + 牌面散布 + 洗牌/翻牌动画
3. **解读结果页**：牌面大图 + 分段解读文字 + 滚动长页面
4. **分享海报**：深色背景 + 牌面 + 金色边框 + 二维码

## 九、风险与应对

| 风险 | 等级 | 应对策略 |
|---|---|---|
| **内容合规** | 🔴 高 | 严禁"算命""预测未来"等措辞，定位为"心灵探索"和"心理启发"。AI 输出增加审核层 |
